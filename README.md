# Flow 

O Flow é uma aplicação web de gerenciamento de tarefas em equipe, desenvolvida com Django e Bootstrap, para auxiliar com css. Ele permite 
que pequenas equipes ou grupos de estudos organizem seus projetos e tarefas em um quadro do tipo kanbam simples, com acompanhamento de
status, comentários e notificações


## Funcionalidades da aplicação 

- Cadastro e autenticação de usuários
- Criação e gerenciamento de projetos
- Kanbam Board com 3 colunas: A Fazer, Em Progresso, Concluído
- Criação e atribuição de tarefas a membros
- Mudança de status direto no board
- Comentários nas tarefas
- Notificações na plataforma
- Histórico de alterações por tarefa
- Gerenciamento de membros por projeto
- Página de perfil com estatísticas

## Tecnologias utilizadas

- Python 3.x
- Django 4.x
- Bootstrap 5
- SQLite
